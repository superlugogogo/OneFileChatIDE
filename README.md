# OneFileChatIde

一个基于单 HTML 文件的轻量级 AI 辅助前端开发 IDE。集成了代码编辑器、实时预览窗口和 LLM 对话窗口，支持通过自定义 API 调用大语言模型辅助生成和修改 HTML/CSS/JS 代码。

## ✨ 介绍

**OneFileChatIde** 旨在提供一个“即开即用”的在线前端开发环境。无需配置复杂的本地开发环境，只需一个 HTML 文件即可运行。

**主要特性：**
*   🚀 **零依赖单文件**：所有逻辑、样式和结构均包含在一个 `.html` 文件中，方便分发和存储。
*   🤖 **AI 辅助编程**：内置 AI 对话窗口，支持流式响应，可自动将生成的代码提交到编辑器。
*   👁️ **实时预览**：左侧代码修改即时反映在预览 iframe 中。
*   💾 **本地持久化**：利用 IndexedDB 本地存储 API Key、系统提示词、历史记录等配置，刷新不丢失。
*   🎨 **语法高亮**：自定义实现了 HTML/CSS/JS 混合代码的高亮显示。
*   📥 **导出下载**：支持将编辑的代码一键下载为 `.html` 文件。

## 🛠️ 技术栈

*   **HTML5 / CSS3**: 使用 Flexbox 布局，深色主题 UI。
*   **Vanilla JavaScript (ES6+)**: 原生 JS 实现，无外部框架依赖。
*   **IndexedDB**: 用于本地存储用户配置和历史对话记录。
*   **Fetch API**: 用于与 LLM API 进行通信，支持 Server-Sent Events (SSE) 流式响应。
*   **Blob & URL.createObjectURL**: 用于实现代码的实时预览和文件下载。

## 📥 安装与运行

由于本项目为单文件应用，无需安装任何依赖或运行构建命令。

1.  下载或复制 `OneFileChatIde.html` 文件。
2.  使用任意现代浏览器（Chrome, Edge, Firefox, Safari）直接打开该 HTML 文件。
3.  在右侧 AI 对话窗口的配置面板中设置你的 LLM API 地址和 Key。

## 💡 简单示例

1.  **配置 API**：
    *   点击右上角 AI 面板的 `+` 号展开配置。
    *   填入兼容 OpenAI 格式的 API 地址（如 `http://127.0.0.1:8080/v1/chat/completions`）。
    *   填入有效的 API Key。
    *   自定义系统提示词（可选）。

2.  **开始对话**：
    *   在输入框中"案例2"按钮输入需求。
    *   点击“发送”或按 `Enter` 键。
    *   AI 将分析需求，通过 `html_commit` 工具将代码写入左侧编辑器，并自动在预览窗口渲染。

3.  **查看历史**：
    *   在配置面板的“历史记录”列表中，点击之前的对话片段即可恢复当时的代码状态和对话上下文。

4.  **下载代码**：
    *   点击左侧代码面板头部的下载图标，将当前代码保存为 `demo.html`。
